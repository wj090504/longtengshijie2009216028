Talk about JavaScript prototype chain

First of all, I would write a sample code.

   function obj(uname){
            this.name = uname;
       }
   var temp = new obj("ck");
   function oldobj(csname){
       this.add="addinfo";
       obj.call(this,csname);
     }
   function newobj(csname){
        this.age = 30;
       oldobj.call(this,csname);
   }
   var csnewobj = new newobj("jack");

When the object access attributes, and if they cannot find inside, then in the prototype object to find the attribute; if the prototype object still could not find the object attribute, prototype prototype object to find their own, and so on, until you find the property or to the top of the elephant. The object by object lookup attributes constitute a chain, called prototype chain.

So,NewObject, oldObject and topObject constitute a prototype chain.
